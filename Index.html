🗄️ Step 1 — Set Up Your Database (JSONBin)

This stores all your subscribers online so both you and Ugochi can see them from any phone or device.
  1. Go to jsonbin.io on your phone → Sign up free
  2. Once inside click CREATE BIN
  3. Paste this inside the bin: [] (just empty square brackets) → click Create
  4. Copy the BIN ID shown in the URL (looks like 64abc123...)
  5. Go to API Keys in the menu → copy your Master Key
  6. Paste both below and click Save ✅
✅ Saved!

🤖 Step 2 — Anthropic API Key (for Newsletter AI)

  1. Go to console.anthropic.com → Sign up
  2. Click API Keys → Create Key → Copy it
  3. Paste below and save ✅
✅ Saved!

📬 Step 3 — Brevo (for Sending Emails)

  1. Go to brevo.com → Sign up free

  4. Go to Settings → API Keys → Copy your key
  5. Paste below and save ✅
